# HUMAN SIMULATE Official Website

Official website repository for the indie game circle **HUMAN SIMULATE**.

## Overview
This repository contains the source files for the official website of HUMAN SIMULATE.

The site is intended to serve as a hub for:
- Game information
- Official links
- Circle profile
- Streaming guidelines
- Future recruitment information

## Planned Pages
- Home
- Works
- About
- Links
- Streaming Guidelines
- Recruit

## Notes
This repository is for the website only.
Game files and other production assets are managed separately.
